# CPool_bistro-matic_2019

The goal was to do a basic calculator for infinite number.  

Date: November 2019

Language: C

Grade: A  

## Compilation:

Compilation via Makefile  
```bash
make
make clean
./calc
```

remove compilation files:
```bash
make fclean
```

## Unit tests

Using Criterion: https://github.com/Snaipe/Criterion

```bash
make tests_run
```
